# Modelo de Negocio — TorneoApp (resumen expandido)

## Propuesta de valor
Plataforma para gestión de torneos y desafíos amateur con foco en equipos locales. Reduce fricción administrativa (fixtures, resultados, cobros) y facilita comunicación.

## Segmento objetivo
- Equipos y organizadores amateur (p. ej. zonas fronterizas como PJC / Ponta Porã).

## Fuentes de ingresos
- Suscripción por equipo (presidente administra el pago).
- Comisiones por transacciones (inscripciones, pagos de multas).
- Publicidad local y patrocinios para torneos.
- Servicios premium: análisis, personalización, soporte prioritario.

## Pricing (ejemplo)
- Plan Básico: gestión de equipo y fixtures (gratuito limitada).
- Plan Pro: $X/mes por equipo — incluye notificaciones, fixtures avanzados, reportes.
- Plan Premium: add-ons: pagos integrados, espacios publicitarios, API privada.

## Go-to-market
1. MVP en una ciudad piloto (contacto con ligas locales).
2. Beta con 50 equipos; recopilar feedback, ajustar UX.
3. Expandir por región con campañas locales y alianzas.

## KPIs clave
- Número de equipos activos
- Retención mensual por equipo
- % de pagos y volumen procesado
- Ingresos MRR

## Roadmap (resumido)
1. MVP: gestión de equipos, fixtures, resultados, registro de usuarios.
2. Autenticación completa y roles (hecho).
3. Pagos y contabilidad básica.
4. Escalado regional y monetización.
